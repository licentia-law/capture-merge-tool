# 📸 캡쳐 병합 도구 (Capture Merge Tool)

윈도우 캡쳐 기능을 사용하여 여러 장의 이미지를 촬영하고, 이를 화질 저하 없이 하나의 세로형 이미지로 간편하게 통합해 주는 도구입니다.

## ✨ 주요 기능

- **실시간 자동 수집**: 프로그램을 실행해 둔 상태에서 윈도우 캡쳐(`Win + Shift + S` 등)를 하면 자동으로 앱 내 캔버스에 이미지가 추가됩니다.
- **무손실 병합**: 캡쳐된 원본 해상도를 그대로 유지하며 PNG 포맷으로 저장하여 화질 저하가 없습니다.
- **간편한 UI**: 실시간 썸네일 미리보기를 제공하며, 스크롤을 통해 수집된 내용을 확인할 수 있습니다.
- **폴더 선택 저장**: 완료 시 사용자가 원하는 위치를 직접 선택하여 결과물을 저장할 수 있습니다.

## 🚀 사용 방법

### 1. 실행 파일 사용 (추천)
- `dist/capture_merged.exe` 파일을 실행합니다.

### 2. 캡쳐 진행
- 프로그램을 켜둔 채로 윈도우 단축키 **`Win + Shift + S`** 또는 **`PrtScn`** 키를 눌러 필요한 부분을 캡쳐합니다.
- 캡쳐할 때마다 프로그램 화면에 자동으로 이미지가 누적되는 것을 확인할 수 있습니다.

### 3. 저장 및 완료
- 모든 캡쳐가 완료되면 우측 상단의 **[완료 및 저장]** 버튼을 클릭합니다.
- 파일을 저장할 폴더를 선택합니다.
- 선택한 폴더에 `capture_merged.png` 파일이 생성되며 완료 팝업이 뜹니다.

## 🛠 개발 및 실행 환경 (소스 코드 실행 시)

### 필수 패키지 설치
```bash
pip install -r requirements.txt
```

### 프로그램 실행
```bash
python capture_merge.py
```

### 빌드 (exe 파일 생성 시)
```bash
pyinstaller --onefile --noconsole --name capture_merged capture_merge.py
```

## 📝 참고 사항
- 프로그램 실행 시 이전에 남아있던 클립보드 이미지로 인한 혼선을 방지하기 위해 클립보드를 한 번 초기화합니다.
- 세로로 길게 이어 붙이는 방식(Vertical Merge)을 기본으로 합니다.
